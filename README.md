# 报销材料智能整理工具

> 将报销凭证拍照/截图发给我，我帮你自动归类整理，生成符合财务规范的报销材料包。

---

## 🎯 我能做什么

当你有一笔报销需要整理时，把所有凭证发给我（发票照片、付款截图、订单截图等），我会自动完成：

- 按报销项目分文件夹
- 按费用项分门别类（每笔费用单独一个文件夹）
- 按凭证类型再细分（发票、订单截图、付款凭证）
- 生成可直接填入U8系统的报销事由文本
- 打包成 .zip 发给你，直接提交财务

---

## 📋 支持的报销类型

### 日常费用
发票 + 付款凭证 + 订单截图 + 费用事由

### 出差费用
日常费用全部 + 出差审批记录截图

---

## 🚀 使用方法

### 第一步：发给我原始凭证

把以下材料发到对话里（顺序不限，命名不限）：

- 📸 发票/收据照片（电子发票PDF也可以）
- 📸 付款凭证截图（银行卡/微信/支付宝）
- 📸 订单截图（美团/饿了么等App，显示买了什么）
- 📸 出差审批记录截图（出差报销需要）
- 📝 费用背景（谁、为什么、在哪、几个人）

### 第二步：我自动处理

我收到材料后会：

1. OCR识别提取关键信息（日期/金额/商户/内容）
2. 主动询问缺失信息（如果有的话）
3. 按规范结构整理文件夹
4. 生成报销事由.txt
5. 打包.zip发给你

### 第三步：检查并提交

- 检查报销事由内容是否准确
- 如有错误告诉我，我立即修改
- 将.zip解压，提交财务

## 📌 常见问题

**Q: 发票模糊看不清怎么办？**
A: 尽量发清晰的照片或电子发票截图，OCR识别会更准确。

**Q: 多笔费用可以一起报销吗？**
A: 可以，分别告诉我每笔费用的背景，我按项目整合。

**Q: 出差补贴怎么填？**
A: 告诉我天数和补贴标准，我在报销事由中单独列一项"差旅补贴"。

**Q: 报销事由里要写哪些内容？**
A: 我会帮你写好完整的背景、发票信息、订单明细、付款凭证、商业目的、U8表单字段，你只需检查准确性。
