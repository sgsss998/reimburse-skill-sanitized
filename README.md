## 🎯 我能做什么

当你有一笔报销需要整理时，把所有凭证发给我（发票照片、付款截图、订单截图等），我会自动完成：

- 按报销项目分文件夹
- 按 U8 费用类型分组（同类型合并、不同类型分开）
- 按凭证类型归档（发票、订单截图、付款凭证等）
- 生成可直接填入U8系统的报销事由文本
- 打包成 .zip 发给你，由你手动提交财务

## 🚀 使用方法

### 第一步：发给我原始凭证

把以下材料发到对话里（顺序不限，命名不限）：

- 📸 发票/收据照片（电子发票PDF也可以）
- 📸 付款凭证截图（银行卡/微信/支付宝）
- 📸 订单截图（美团/饿了么等App，显示买了什么）
- 📸 出差审批记录截图（出差报销需要）
- 📝 费用背景（谁、为什么、在哪、几个人）

### 第二步：我自动处理

我收到材料后会：

1. OCR识别提取关键信息（日期/金额/商户/内容）
2. 主动询问缺失信息（如果有的话）
3. 发票图片统一转 PDF（截图凭证保留原格式）
4. 按规范结构整理文件夹
5. 生成报销事由.txt
6. 打包.zip发给你

### 第三步：检查并提交

- 检查报销事由内容是否准确
- 如有错误告诉我，我立即修改
- 将.zip解压，**由你手动提交财务**（Skill 不会自动代提交）

## 📌 常见问题

**Q: 发票模糊看不清怎么办？**
A: 尽量发清晰的照片或电子发票截图，OCR识别会更准确。

**Q: 多笔费用可以一起报销吗？**
A: 可以。同一 U8 类型会合并列示，不同 U8 类型会分开处理。

**Q: 出差补贴怎么填？**
A: 以你们实际制度与审批单为准；我只按你提供的凭证和指令写入。

**Q: 报销事由里要写哪些内容？**
A: 我会按“逐笔费用明细 + 商业目的 + 备注 + U8表单字段 + 附件清单”输出，你只需校对后提交。
