## 我能做什么

- 按报销项目建档
- 按 U8 费用类型分组整理
- 自动生成 `报销事由.txt`
- 打包 `.zip` 交付

---

## 使用方式

1. 把 `报销.skill` 放进你的 Agent skills 目录。  
2. 触发关键词：`报销`、`报销整理`、`报销材料`、`贴发票`。  
3. 上传发票、付款凭证、订单截图、审批记录等材料。  
4. Agent 输出规范化目录与报销事由文本。

## 安全说明

本仓库为公开版本，默认执行保守脱敏策略：

- 真实人名、公司名、税号、账号、电话、详细地址 → `[已脱敏]` 或通用占位词
- 具体出行地、商户细节、商品细节、报销额度 → 通用表达
- 不确定是否敏感的信息 → 默认脱敏
